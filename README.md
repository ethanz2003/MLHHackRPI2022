# MLHHackRPI2022

## How to Run
1. Clone the repository onto your system.
2. Install the OpenCV python packages used by the script (recommended in a virtual environment).
   - pip install opencv-python
   - pip install cvzone
3. Open the Unity Project through Unity Hub.
4. Execute the Python script coordinates.py through command line or vscode run.
5. Run the Unity project in the Unity Editor.
